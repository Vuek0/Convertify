# 🖼️ Convertify

Современный веб-конвертер файлов с красивым анимированным интерфейсом.

**Демо:** [convertify.uz](https://convertify.uz)
**GitHub:** [github.com/Vuek0/Convertify](https://github.com/Vuek0/Convertify)

## 🔐 Безопасность

**Convertify полностью безопасен:**

- 📁 **Файлы не хранятся** — удаляются сразу после конвертации
- 🔒 **Никакого отслеживания** — не собираем персональные данные
- 🌐 **Vercel Serverless** — обработка в ephemeral-среде
- 💻 **Open Source** — проверьте код сами

> **Как это работает:** Вы загружаете файл → он временно сохраняется в изолированной serverless-среде → конвертируется → отправляется вам → **немедленно удаляется**. Никакого постоянного хранилища.

[Политика конфиденциальности](https://convertify.uz/privacy)

## ✨ Особенности

- 🎨 **Современный UI** — градиенты, анимации, стеклянный эффект
- 🌓 **Тёмная/светлая тема** — переключатель с сохранением выбора
- 🎭 **Плавные анимации** — кастомный select с каскадным появлением опций
- 📱 **Адаптивность** — работает на любых устройствах
- ☁️ **Drag & Drop** — загрузка файлов перетаскиванием
- ⚡ **Быстрая конвертация** — обработка на сервере за секунды
- 🔍 **SEO-оптимизирован** — meta-теги, Schema.org, sitemap

## 🚀 Реализованный функционал

### Интерфейс
| Компонент | Описание |
|-----------|----------|
| 🌓 Тема | Переключатель светлой/тёмной темы с сохранением в localStorage |
| 📤 Загрузка | Drag & Drop зона + клик для выбора файла |
| 👁️ Предпросмотр | Отображение имени и размера выбранного файла |
| 🎯 Select | Анимированный выбор формата с кастомным дизайном |
| 🔄 Индикация | Анимация загрузки при конвертации |
| 📱 Адаптивность | Мобильная и десктопная версии |

### Конвертация
| Входной формат | Выходные форматы | Описание |
|----------------|------------------|----------|
| **PNG, JPG, JPEG** | PDF, PNG, JPEG, WebP, ICO, TIFF | Конвертация изображений |
| **PDF** | PNG, JPEG, DOCX | PDF в изображения или Word |
| **DOCX** | PDF, PNG, JPEG | Word в PDF или изображения |

### Технические особенности
- ✅ Валидация загружаемых файлов (PNG, JPG, JPEG, PDF, DOCX)
- ✅ Генерация уникальных имён для временных файлов
- ✅ Автоматическая очистка временных файлов после конвертации
- ✅ Правильные MIME-типы для скачивания
- ✅ Обработка ошибок с понятными сообщениями

## 🏃 Локальный запуск

### Требования
- Python 3.8 или выше
- pip

### Установка

```bash
# 1. Клонируйте репозиторий
git clone <ваш-репозиторий>
cd pngtopdf

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте окружение
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Запустите приложение
python app.py
```

### Использование

Откройте в браузере: **http://localhost:5000**

1. Перетащите файл или кликните для выбора
2. Выберите нужный формат вывода
3. Нажмите «Конвертировать»
4. Файл автоматически скачается
